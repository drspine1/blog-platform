## Qu'est-ce que le SEO On-Page ?

Le SEO on-page désigne toutes les optimisations que vous effectuez directement sur une page web pour améliorer ses classements dans les moteurs de recherche. Contrairement au SEO off-page (backlinks) ou au SEO technique (infrastructure du site), le SEO on-page est entièrement sous votre contrôle.

Faites-le correctement et vous donnez à chaque contenu les meilleures chances possibles de se classer.

## 1. Balises de Titre

La balise de titre est l'élément SEO on-page le plus important. Elle apparaît comme le titre cliquable dans les résultats de recherche.

### Règles :
- **Longueur** : 50 à 60 caractères (plus long est tronqué)
- **Incluez votre mot-clé principal** — idéalement au début
- **Rendez-la convaincante** — c'est aussi une publicité pour votre page
- **Unique pour chaque page** — ne dupliquez jamais les balises de titre

### Exemples :
```
❌ Accueil | Mon Site
✅ Guide de Recherche de Mots-Clés : Trouvez des Mots-Clés qui Convertissent (2024)
```

## 2. Méta-Descriptions

Les méta-descriptions n'affectent pas directement les classements, mais influencent fortement le taux de clics — ce qui, lui, a un impact.

### Règles :
- **Longueur** : 150 à 160 caractères
- **Incluez votre mot-clé principal** — Google le met en gras dans les résultats
- **Rédigez une vraie proposition de valeur** — qu'obtiendra le lecteur ?
- **Incluez un appel à l'action doux** — « Apprenez comment... », « Découvrez... »

## 3. Structure des En-têtes

Les en-têtes (H1 à H6) aident à la fois les utilisateurs et les moteurs de recherche à comprendre la hiérarchie de votre contenu.

### Règles :
- **Un seul H1 par page** — c'est le titre de votre page, faites-en sorte qu'il compte
- **H2 pour les sections principales**, H3 pour les sous-sections
- **Incluez les mots-clés naturellement** — ne les forcez pas
- **Rendez les en-têtes descriptifs** — un utilisateur devrait comprendre la structure de la page rien qu'avec les en-têtes

## 4. Structure des URL

Des URL propres et descriptives sont meilleures pour le SEO et les utilisateurs.

```
❌ votresite.com/p?id=1234
❌ votresite.com/blog/2024/03/15/mon-premier-article-sur-le-seo
✅ votresite.com/blog/guide-seo-on-page
```

### Règles :
- Utilisez des tirets, pas des underscores
- Gardez-les courtes et descriptives
- Incluez votre mot-clé principal
- Utilisez uniquement des minuscules

## 5. Qualité et Profondeur du Contenu

Le système de contenu utile de Google récompense le contenu qui aide vraiment les utilisateurs. Le contenu mince et générique est rétrogradé.

### Ce qui fait classer le contenu :
- **Exhaustivité** — couvrez le sujet en profondeur
- **Originalité** — ajoutez vos propres insights, données ou perspective
- **Exactitude** — citez des sources, gardez les informations à jour
- **Lisibilité** — paragraphes courts, sous-titres, points de liste
- **E-E-A-T** — démontrez l'Expérience, l'Expertise, l'Autorité, la Fiabilité

## 6. Optimisation des Mots-Clés

### Placement du mot-clé principal :
- Balise de titre
- En-tête H1
- Dans les 100 premiers mots de l'introduction
- Au moins un sous-titre H2
- Méta-description
- Slug d'URL
- Texte alternatif des images (si pertinent)

### Mots-clés secondaires et termes LSI :
Parsemez des termes connexes naturellement dans tout le texte. Ne les bourrez pas — écrivez d'abord pour les humains.

## 7. Optimisation des Images

Les images affectent à la fois la vitesse de page (un facteur de classement) et l'accessibilité.

### Checklist :
- [ ] Noms de fichiers descriptifs (`checklist-seo.webp` pas `IMG_1234.jpg`)
- [ ] Texte alternatif sur chaque image — décrivez ce qui est dans l'image, incluez le mot-clé si naturel
- [ ] Compressez les images (utilisez le format WebP)
- [ ] Définissez des attributs de largeur et hauteur explicites (prévient le CLS)
- [ ] Chargement différé des images sous le pli

## 8. Liens Internes

Les liens internes distribuent l'autorité de page sur votre site et aident les utilisateurs à découvrir du contenu connexe.

### Meilleures pratiques :
- Créez des liens vers des pages pertinentes en utilisant un texte d'ancre descriptif
- Visez 3 à 5 liens internes par article
- Assurez-vous que vos pages les plus importantes reçoivent le plus de liens internes
- Corrigez immédiatement les liens internes brisés

### Ce qu'il faut éviter :
- Texte d'ancre générique (« cliquez ici », « en savoir plus »)
- Lier plusieurs fois à la même page dans un article
- Pages orphelines (pages sans liens internes pointant vers elles)

## 9. Balisage de Schéma

Le balisage de schéma (données structurées) aide Google à comprendre votre contenu et peut débloquer des résultats enrichis dans les recherches.

### Types de schéma les plus précieux pour les blogs :
- **Article / BlogPosting** — pour les articles de blog
- **FAQPage** — pour les sections FAQ
- **HowTo** — pour les guides étape par étape
- **BreadcrumbList** — pour la navigation du site

Utilisez le format JSON-LD et testez avec l'outil de test des résultats enrichis de Google.

## 10. Signaux d'Expérience de Page

Au-delà du contenu, Google considère :
- **Core Web Vitals** (LCP, INP, CLS)
- **Compatibilité mobile** — testez avec l'outil de test mobile de Google
- **HTTPS** — les pages non sécurisées sont signalées dans Chrome
- **Pas d'interstitiels intrusifs** — les pop-ups qui bloquent le contenu sur mobile

## Le Processus d'Audit SEO On-Page

Effectuez cet audit sur chaque page importante trimestriellement :

1. Vérifiez la balise de titre et la méta-description dans Search Console (données CTR)
2. Examinez la structure des en-têtes
3. Vérifiez le placement des mots-clés
4. Vérifiez les liens internes (ajoutez ceux qui manquent)
5. Testez la vitesse de page dans PageSpeed Insights
6. Validez le balisage de schéma
7. Examinez le contenu pour l'exactitude et la fraîcheur

Le SEO on-page n'est jamais « terminé » — c'est un processus continu de raffinement basé sur les données de performance.
